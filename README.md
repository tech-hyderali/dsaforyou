# DSA For You

A comprehensive platform for learning Data Structures and Algorithms (DSA) with a focus on simplicity and practical understanding.

## Features

- **Learn**: Structured learning paths for mastering DSA concepts
- **Practice**: Curated collection of problems with varying difficulty levels
- **Projects**: Hands-on projects to apply DSA knowledge
- **Notes**: Personal note-taking system for DSA concepts

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- React Router DOM
- Lucide React Icons

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/dsaforyou.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm run dev
   ```

## Project Structure

```
src/
├── components/     # Reusable UI components
├── context/       # React context providers
├── hooks/         # Custom React hooks
├── pages/         # Main application pages
└── types/         # TypeScript type definitions
```
